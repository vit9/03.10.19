#  API

## Установка и настройка

##### Миграция структуры базы

> Чтобы выполнить миграцию и создать все таблицы и поля в базе данных выполните следующую команду

```bash
yarn db:migrate
```

##### Заполнение демо данных в базе


##### Откат демо данных

> Чтобы откатить и убрать все демо данные, выполните следующую команду

```bash
yarn db:migrate:refresh
```

##### Запуск

```bash
node server.js
```
